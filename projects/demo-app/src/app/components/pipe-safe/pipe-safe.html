SafePipe

Angular's DomSanitizer automatically strips tags like <mark> to prevent XSS. The SafePipe allows these elements to render by marking the input string as trusted.

Import import {{ '{' }} SafePipe {{ '}' }} from 'dhemant-utils'; Registration imports: [ SafePipe ] Input Value '<mark>Trusted Highlight</mark>' Usage <div [innerHTML]="htmlString | dhSafe: 'html'"></div>

How It Works

  • Raw Input: Accepts a standard string containing HTML, CSS, or URLs.
  • Security Bypass: It wraps the string in a SafeValue object using DomSanitizer.
  • Rendering: Angular detects the SafeValue and skips its internal sanitization logic.

Allowed Context Options

Type Bypasses... Example Use Case
'html' HTML Sanitizer Rendering <mark> or <svg> tags.
'style' CSS Sanitizer Dynamic gradients in [style] bindings.
'resourceUrl' Resource URL External src for <iframe>.
'url' URL Sanitizer Dynamic links in <a href>.
