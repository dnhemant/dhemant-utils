Click Outside Directive

Essential for managing Floating UI like dropdowns. It captures clicks outside the host element to trigger closure logic.

Import import {{ "{" }} ClickOutsideDirective {{ "}" }} from 'dhemant-utils'; Registration imports: [ClickOutsideDirective] Selector [dhClickOutside] Usage <div dhClickOutside (clickOutside)="onClose()">...</div>

How it works

  • Global Listener: Attaches a listener to the document level using Angular's host listener or Renderer2.
  • Boundary Logic: Emits only if the event.target is determined to be outside the DOM hierarchy of the host element.
