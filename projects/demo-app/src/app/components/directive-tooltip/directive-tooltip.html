Tooltip Directive

A lightweight utility to display contextual text hints on hover. Now supports dynamic positioning and custom CSS theming through simple input properties.

Import import {{ "{" }} TooltipDirective {{ "}" }} from 'dhemant-utils'; Registration imports: [TooltipDirective] Inputs
[tooltipPosition]: 'top' | 'bottom' | 'left' | 'right' [tooltipClass]: string (custom CSS class)
Usage <button dhTooltip="Hi!" tooltipPosition="right">...</button>

Advanced Features

  • Smart Positioning: Automatically calculates viewport coordinates to prevent clipping and ensures the tooltip aligns with the chosen axis.
  • Styling Hooks: Use tooltipClass to pass any global CSS class for custom colors, fonts, or shadows.
  • Safe Cleanup: Uses Renderer2 to safely remove elements from the DOM, ensuring compatibility with Angular Universal.
